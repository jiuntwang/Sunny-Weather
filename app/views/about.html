About Sunny Weather App

A weather application assignment created for WATS4000 at Seattle University by Jiun Wang (2017)

Tools

  • Using AngularJS, a Single Page Application Framework (SPAF)
  • Added Bootstrap and SASS
  • Data from the Open Weather Map" API

Features

  • Search for Weather by City or Zip Code
  • Will display a search query list if multiple city names exist
  • Ability to view current temperature, high and low, wind, and cloud conditions
  • Can check the 7-day weather forecast
  • Save cities to a list on the Homepage
